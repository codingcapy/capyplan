# capyplan

GenAI Financial plan under construction. Create financial plans and generate recommendations using AI.

Live at: https://capyplan.up.railway.app/

To install backend dependencies:

```bash
bun i
```

To run backend:

```bash
bun start
```

To install frontend dependencies:

```bash
cd frontend
bun i
```

To run frontend:

```bash
cd frontend
bun dev
```

You will need the appropriate environment variables for the app to work.

This project was created using `bun init` in bun v1.3.3. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
